## OVERVIEW
This analysis examined 50,000 maternal health records to understand the 
relationship between comorbidity burden and key demographic and healthcare 
access factors.

## 4. COUNT MODEL SELECTION & JUSTIFICATION

Dependent Variable: Comorbidity count (0-8 conditions)
  • Mean = 0.91, Variance = 0.995
  • Dispersion ratio = 1.09 (variance slightly > mean)
  
Model Comparison:
  • Poisson AIC = 127,020
  • Negative Binomial AIC = 126,850
  • Difference = 170 points (STRONG evidence for NB)
  
Selected Model: NEGATIVE BINOMIAL REGRESSION
Justification:
  ✓ Overdispersion present (though modest)
  ✓ Negative binomial provides better fit (170-point AIC improvement)
  ✓ Negative binomial is more conservative/robust
  ✓ Avoids underestimating standard errors (Poisson assumption violation)

## 5. MODEL RESULTS (Key Findings)

Predictors with Statistical Significance (p < 0.05):

1. **Comprehensive Postnatal Care (p = 0.0003)**
   IRR = 1.039 [95% CI: 1.018, 1.060]
   → 3.9% INCREASE in expected comorbidity count
   
   ⚠️ CRITICAL INTERPRETATION:
   This counterintuitive finding likely indicates REVERSE CAUSALITY or 
   confounding. Women with more comorbidities may receive MORE postnatal 
   care, not less. This does NOT suggest postnatal care causes 
   comorbidities. To disentangle causality would require:
     • Longitudinal data tracking care before outcome
     • Instrumental variables
     • Randomized controlled trial design

2. **Age (p < 0.0001)**
   IRR = 1.016 [95% CI: 1.0146, 1.0177] per year
   → 1.6% increase in expected comorbidities per additional year
   → STRONG, consistent biological relationship
   → At age 44 vs 25: expected count increases from ~0.86 to ~1.11

3. **Insurance: State-Provided vs No Insurance (p = 0.028)**
   IRR = 0.972 [95% CI: 0.947, 0.997]
   → 2.8% DECREASE in expected comorbidities
   → Modest protective effect; may reflect selection effects or access

4. **Education Effects (p < 0.05 for multiple categories)**
   Some College vs College:  IRR = 0.938 (p = 0.0002)
   High School vs College:   IRR = 0.959 (p = 0.012)
   Post-Grad vs College:     IRR = 0.929 (p = 0.024)
   → Pattern: Lower education → higher comorbidity burden
   → May reflect SES gradient in health outcomes

Non-Significant Predictors:
  • Distance to provider (p = 0.52)
  • Race/ethnicity main effects (p > 0.3 for all)
  • Private vs no insurance (p = 0.13)

## 6. MODEL ASSUMPTIONS & DIAGNOSTICS

Negative Binomial Assumptions:
  ✓ Count outcome: Verified (0-8 discrete counts)
  ✓ Independence: Assumed (each woman one observation)
  ✓ Conditional mean-variance relationship: Verified via theta = 10.18
  ✓ No zero-inflation detected: Observed zeros (42.7%) align with 
    predicted distribution

Model Fit Diagnostics:
  ✓ Residual deviance (56,264) < Null deviance (56,712)
  ✓ Improvement = 448 points (substantial)
  ✓ Diagnostic plots show no severe patterns
  ✓ Large sample (N=50,000) provides robust inference

Limitations:
  ⚠ Cross-sectional data: Cannot establish causality
  ⚠ Synthetic data: Patterns may not reflect real-world relationships
  ⚠ Unmeasured confounding: Many factors (genetics, comorbidity severity)
    not included
  ⚠ Distance variable quality: Extreme values suggest measurement issues
  ⚠ Reverse causality: Postnatal care effect likely reflects treatment
    response to higher-risk women

## 7. RESPONSIBLE REPORTING

What This Analysis SHOWS:
  • Age is the strongest demographic predictor of comorbidity burden
  • Education and insurance show modest associations with comorbidities
  • Postnatal care is associated with comorbidity count

What This Analysis DOES NOT SHOW:
  ✗ That postnatal care CAUSES comorbidities (reverse causality likely)
  ✗ That distance to provider is irrelevant (may be mediated by insurance)
  ✗ Causal effects of any kind (cross-sectional design)
  ✗ Individual-level risk prediction (group-level inference only)

Appropriate Use:
  • Hypothesis generation for future studies
  • Descriptive characterization of sample
  • Methodological example of count modeling
  • NOT for clinical decision-making or policy without validation
