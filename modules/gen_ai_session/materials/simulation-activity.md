# Simulation Activity

## Goal

Infer the structure of a dataset from a practice file or schema and outline a simple synthetic data plan.

## Look For

- variable names
- variable types
- plausible ranges
- missingness patterns
- likely relationships between variables
- possible outcomes or grouping variables

## Output

Create a short note that describes:

1. the variables you identified
2. the distributional features you would preserve
3. one or two relationships you would preserve
4. how you would use the synthetic data safely in a workflow

## Reminder

The goal is to preserve structure for workflow development, not to recreate real people.
