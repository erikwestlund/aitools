# Instructor Notes

## Core Framing

- practical over abstract
- workflow over hype
- safety and reproducibility are core content
- verification before trust

## Session Shape

- short lecture
- short demo
- hands-on activity
- repeat three times

## Large-Class Notes

- expect account and authentication friction
- encourage pair work if troubleshooting slows momentum
- keep one backup path for students who cannot push live
- use only synthetic or public files in demonstrations

## Key Refrain

Code can move across environments; sensitive data cannot.
