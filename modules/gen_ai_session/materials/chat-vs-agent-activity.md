# Chat vs Agent Activity

## Goal

Compare what happens when you use a chat workflow versus an agent workflow for a small analytical task.

## Record the Following

- what task you asked for
- what context you provided
- what the tool produced
- one thing that seems useful
- one thing that needs verification
- one safety or privacy concern

## Discussion Prompt

Be ready to explain:

- when chat is enough
- when an agent is worth the extra setup
- what could go wrong if you trusted the output too quickly
