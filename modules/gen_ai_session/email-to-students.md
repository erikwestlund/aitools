Hi everyone,

Before Friday's session, please try to set up a few things in advance so we can spend more time working and less time troubleshooting.

First, please make sure you have a GitHub account. If you do not already have one, you can register here: https://github.com/signup

Second, please install `git` on your computer and make sure you can run it from a command line.

Mac users:

- open the `Terminal` app and run `git --version`
- if prompted, install the Xcode Command Line Tools (this step can take a little while, so please do not leave it until right before class)

Windows users:

- install Git for Windows from https://git-scm.com/download/win
- to check that Git is working, open `PowerShell` or `Git Bash` (which comes with the Windows git installer) and run `git --version`

Installing tools and registering accounts during class and troubleshooting setup can take a lot of time, so doing this ahead of time will help a lot.

We will also demo some agent-based AI workflows. If you want to try those yourself, it is helpful to have access to at least one of the following:

- ChatGPT
- Claude
- GitHub Copilot / GitHub Education

GitHub Copilot has a student offering through GitHub Education: https://github.com/education

It is useful, but it does not currently provide access to some of the strongest models, including GPT-5.4, Claude Opus, and Claude Sonnet. It can still be enough to try some agent-style workflows.

Free chat tools such as HopGPT are also fine, but you should expect to be limited mostly to chat-based workflows rather than full agent workflows.

Thank you!
